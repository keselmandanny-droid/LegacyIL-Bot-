# פקודות עברית - Hebrew Commands

## סקירה כללית
תיקייה זו מכילה את כל פקודות הבוט מתורגמות לעברית כוללות:

- **Descriptions** - תיאורי פקודות
- **Option names** - שמות אפשרויות
- **Option descriptions** - תיאורי אפשרויות
- **Embed titles and messages** - כותרות וטקסט הודעות
- **Help text** - טקסט עזרה

## קטגוריות

### קטגוריית Core (ליבה)
- **ping.js** - בדיקת זמן הפעלה
- **help.js** - תפריט עזרה

### קטגוריית Economy (כלכלה)
- **balance.js** - בדיקת כסף

## שימוש

להעתקת הפקודה לתיקיית הפקודות בפועל:

```bash
cp src/commands_hebrew/*.js src/commands/[CategoryName]/
```

## שפות נתמכות

- עברית (he-IL)
- English (en-US)

## הערות

- כל הפקודות שומרות על קוד ה-API המקורי
- רק הנושא של ה-UI מתורגם
- ה-IDs של הפקודות נשארים בעברית לקלות שימוש עבור משתמשים ישראלים
